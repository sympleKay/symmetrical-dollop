Question 1: Distinguish clearly between Internet and Web

Answer:

Definition:
The internet is a global network of interconnected computers and other devices that communicate using standardized protocols (like TCP/IP).

Purpose:
It provides the infrastructure for various services like email, file sharing, online gaming, VoIP, and the web.

Examples of services on the internet:

  • Email (e.g., Gmail)
  • File transfer (e.g., FTP)
  • Messaging apps (e.g., WhatsApp, Telegram)
  • Streaming (e.g., Netflix)
  • Remote access (e.g., VPNs)
  • The web (websites, browsers, etc.)

2. The Web (World Wide Web)

Definition:
The web is a service that operates on the internet. It consists of billions of web pages and web applications accessed through web browsers (like Chrome, Safari, Firefox).

Examples of web resources:

  • Websites (e.g., Wikipedia, Facebook)
  • Online stores (e.g., Amazon)
  • Blogs
  • Web apps (e.g., Google Docs)
